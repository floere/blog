Reload a running Ruby application (revisited)

Reload a running Ruby app:

If you need to rename the load_application method, then


class << self
  alias new_load_application load_application
end

and reload, then remove the alias, and reload again. (Just try it if you don’t get it right now)

Used in production, by the way. One needs to be a bit cautious though…
