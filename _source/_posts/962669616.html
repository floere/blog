Ruby Challenge: Merging Ranges

This is a little fun challenge for your brain cells.

A. Given an arbitrary number of Ranges, e.g. (1..9), (5..15), (17..23).

1. Devise an algorithm that merges overlapping and enclosed Ranges such that only independent Ranges (such as (17..23) in the example above) remain.
2. Can you do it in O(n).
3. If not, what’s x in O(x)?

A small example to use as a test case:
ary = [(17..23),(2..4),(1..9),(5..13)]
merge_ranges(ary).should == [(1..13), (17..23)]

How would you do it?
